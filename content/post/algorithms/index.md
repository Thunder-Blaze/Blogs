# **Algorithms & Properties**

### Possible sum of K distinct integers between 1 and N
let the sum be S,
we can form all numbers till $\dfrac{N(N+1)}{2}$
but if we are to use fixed K digits, the Range will be limited to $(K)(N)+(K-1)$
